# MISSION--3-PRODESK-IT-SOLUTIONS-
# Dev Detective — GitHub User Search App

A modern SaaS-style web application that allows users to search for any GitHub profile and view detailed information along with their latest repositories.

---

## 🚀 Live Demo
👉 https://sriniketh-vangipuram.github.io/MISSION--3-PRODESK-IT-SOLUTIONS-/

---

## ✨ Features

### 🔎 User Search
- Search any GitHub username
- Press Enter or click Search

### 👤 Profile Information
- Avatar
- Name / Username
- Bio (with fallback if not available)
- Join Date (formatted)
- Portfolio Link (with validation)

### 📦 Repository Section
- Top 5 latest repositories
- Clickable links to GitHub repos
- Sorted by latest created

### ⚡ UX Enhancements
- Loading state (spinner)
- Error handling (User not found)
- Modal popup for missing data
- Smooth animations and transitions

## 🧪 Edge Cases Handled

- Invalid username → Shows error message
- Empty input → No API call
- Missing bio → Displays "No bio found"
- Missing portfolio → Shows modal instead of opening blank link
